v1.0.0
    LangChain 发布了 1.0 版本，其中包含两项重大变更：
    对 langchain 中的所有链（chain）和代理（agent）进行了全面重构。现在，所有链和代理都被替换为一个统一的、高层次的抽象层：这个抽象层是`基于 LangGraph 构建`的代理抽象层。这个抽象层最初是在 LangGraph 中创建的，后来被移植到了 LangChain 中。
    对于那些仍在使用旧版 LangChain 链/代理、且不愿意升级的用户（请注意：我们强烈建议您升级），您可以通过安装 langchain-classic 包来继续使用旧版的 LangChain。
## 从此langchain就是高于langgraph的抽象层了，langgraph是langchain的一个组件，专注于图形结构和关系，而langchain则提供了更高层次的抽象，支持更复杂的代理系统构建。
2026-03-15
v0.5.3
    Deep Agents 是一个基于 LangGraph 构建的开源代理框架。
    虽然 LangChain 为自定义代理架构提供了灵活的开发工具，但 Deep Agents 为需要长时间运行的复杂任务（如研究和编程）提供了“开箱即用”的解决方案。它内置了规划工具、支持可插拔后端（内存存储、磁盘存储、LangGraph 存储、沙箱环境）的虚拟文件系统，以及用于实现上下文隔离的子代理机制。如果您希望使用预定义的工具来构建更自主的代理系统，可以选择 Deep Agents；如果您希望完全控制代理架构的细节，则可以选择 LangChain。
